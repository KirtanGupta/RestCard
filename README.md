# 🍽️ GD's Fast Food — Digital Menu
A **premium, mobile-first digital menu** web application built for QR code scanning.
Customers scan a QR code and instantly see a beautiful fullscreen image gallery
of the restaurant menu — no app download, no login, no friction.
## ✨ Features
- 📱 **QR-ready** — scan and instantly view the menu
- 🖼️ **Fullscreen gallery** — 8 menu pages displayed as full-screen images
- 👆 **Touch & swipe** — native swipe gesture support (Swiper.js)
- ⌨️ **Keyboard navigation** — arrow keys for desktop users
- ⬅️ ➡️ **Custom nav buttons** — `< >` gold-styled buttons on both sides
- ⚡ **Fast loading** — all images eagerly preloaded, optimized via `next/image`
- 🌓 **Fully responsive** — portrait, landscape, mobile, tablet, desktop
- 💛 **Luxury design** — black & gold theme, Cinzel typography, glassmorphism UI
- 🔄 **Loading splash screen** — animated logo before gallery appears
- 🔢 **Page counter** — shows current page (e.g. `1 / 8`)
